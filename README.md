# MEMORA
A smart Chrome extension for sharing conversation context across AI platforms by summarizing it and  allows to save chat history, select particular history provide to another AI tool , delete particular chat history


# Features

- Automatic conversation context capture
- Smart AI-powered context summarization
- One-click enable/disable from Chrome toolbar
- Seamless context transfer across AI platforms
- Local-first privacy (data stays on your device)
- Works with ChatGPT, Claude, Gemini, and Perplexity
- Lightweight and user-friendly Chrome extension

# Technologies Used

- JavaScript
- HTML
- CSS
- Chrome Extension API
- AI-based Context Summarization
- Local Storage

# Files Used

- manifest.json → Chrome extension configuration
- background.js → Background service worker
- content-chatgpt.js → ChatGPT integration
- content-claude.js → Claude integration
- popup.html → Extension popup interface
- popup.js → Popup functionality
- styles.css → User interface styling
- icons/ → Extension icons

# Learning Outcomes

- Chrome extension development
- Browser automation
- DOM manipulation
- AI context summarization
- Cross-platform integration
- JavaScript event handling
- Local storage management

# Project Outcome

- Eliminates repeated explanations across AI platforms
- Saves time while switching between AI tools
- Improves productivity with seamless context sharing
- Provides a smooth and uninterrupted AI experience
- Keeps user data private with local-first storage
